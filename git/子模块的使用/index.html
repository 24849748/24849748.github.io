【Git】子模块 submodule 的使用

子模块是 git 仓库中的仓库，在一些大型工程中比较常见。
对于 git 来说，子模块仅是主仓库管理的一个指向子仓库某个提交的指针。
虽然在物理层面上来说子模块的代码、文件都在主仓库里，但子模块的 git 管理是独立的。
